Encryption Bot Documentation

Welcome to the Encryption Bot! This bot allows users to apply various encryption and hashing methods to their messages, including Caesar Cipher, Base64 encoding, Morse code, and more. Additionally, users can combine multiple methods to create complex, layered encryption, or apply randomized ciphers and hashing with salts for secure data obfuscation.

Below is the complete documentation on how to use each command, along with examples.

1. General Info

  • Prefix: !
  • Supported Commands: Various encryption, decryption, and hashing methods.
  • Combo Support: You can combine multiple encryption or decryption methods in one go using the combo command.

2. Command Overview

2.1 Encryption and Decryption Commands

  • Morse Code Encryption: !me <message>
  • Morse Code Decryption: !md <morse_code>
  • Caesar Cipher Encryption: !ce <message>
  • Caesar Cipher Decryption: !cd <encrypted_message>
  • Binary Encryption: !be <message>
  • Binary Decryption: !bd <binary_code>
  • Base64 Encryption: !he <message>
  • Base64 Decryption: !hd <encoded_message>
  • Vigenère Cipher Encryption: !ve <key> <message>
  • Vigenère Cipher Decryption: !vd <key> <encrypted_message>
  • ROT13 Cipher: !rot13 <message>
  • Reverse Text: !reverse <message>
  • Hashing (MD5): !hashmd5 <message>
  • Hashing (SHA-256): !hashsha256 <message>
  • Random Substitution Cipher: !rs <message>
  • Salted Hashing: !sh <message>

3. Command Details and Examples

3.1 Morse Code Encryption: !me

Description: Converts a message into Morse code, where each letter or number is represented by a unique combination of dots and dashes.

Usage: !me Hello World

Example Output:

**Morse Code Encryption**
Text: Hello World
Morse Code: .... . .-.. .-.. --- / .-- --- .-. .-.. -..

3.2 Morse Code Decryption: !md

Description: Decodes Morse code back into readable text.

Usage: !md .... . .-.. .-.. --- / .-- --- .-. .-.. -..

Example Output:

**Morse Code Decryption**
Morse Code: .... . .-.. .-.. --- / .-- --- .-. .-.. -..
Decoded Message: Hello World

3.3 Caesar Cipher Encryption: !ce

Description: Encrypts the message using the Caesar Cipher with a default shift of 3.

Usage: !ce Hello

Example Output:

**Caesar Cipher Encryption**
Text: Hello
Encrypted Message: Khoor

3.4 Caesar Cipher Decryption: !cd

Description: Decrypts a Caesar Cipher encrypted message.

Usage: !cd Khoor

Example Output:

**Caesar Cipher Decryption**
Encrypted Message: Khoor
Decrypted Message: Hello

3.5 Binary Encryption: !be

Description: Converts the message into binary code.

Usage: !be Hello

Example Output:

**Binary Encryption**
Text: Hello
Binary Message: 01001000 01100101 01101100 01101100 01101111

3.6 Binary Decryption: !bd

Description: Converts binary code back into a readable message.

Usage: !bd 01001000 01100101 01101100 01101100 01101111

Example Output:

**Binary Decryption**
Binary Message: 01001000 01100101 01101100 01101100 01101111
Text: Hello

3.7 Base64 Encryption: !he

Description: Encodes the message in Base64 format.

Usage: !he Hello

Example Output:

**Base64 Encryption**
Text: Hello
Encoded Message: SGVsbG8=

3.8 Base64 Decryption: !hd

Description: Decodes a Base64 encoded message.

Usage: !hd SGVsbG8=

Example Output:

**Base64 Decryption**
Encoded Message: SGVsbG8=
Decoded Message: Hello

3.9 Vigenère Cipher Encryption: !ve

Description: Encrypts the message using the Vigenère cipher. The user must supply a key for encryption.

Usage: !ve key Hello

Example Output:

**Vigenère Cipher Encryption**
Text: Hello
Encrypted Message: Rijvs

3.10 Vigenère Cipher Decryption: !vd

Description: Decrypts a message encrypted using the Vigenère cipher.

Usage: !vd key Rijvs

Example Output:

**Vigenère Cipher Decryption**
Encrypted Message: Rijvs
Decrypted Message: Hello

3.11 ROT13 Cipher: !rot13

Description: Encrypts or decrypts a message using the ROT13 cipher (shifts the letters by 13 positions).

Usage: !rot13 Hello

Example Output:

**ROT13**
Text: Hello
Transformed Message: Uryyb

3.12 Reverse Text: !reverse

Description: Reverses the given text.

Usage: !reverse Hello

Example Output:

**Reverse Text**
Text: Hello
Reversed Message: olleH

3.13 Hashing (MD5): !hashmd5

Description: Generates an MD5 hash of the input message.

Usage: !hashmd5 Hello

Example Output:

**MD5 Hash**
Text: Hello
MD5 Hash: 8b1a9953c4611296a827abf8c47804d7

3.14 Hashing (SHA-256): !hashsha256

Description: Generates a SHA-256 hash of the input message.

Usage: !hashsha256 Hello

Example Output:

**SHA-256 Hash**
Text: Hello
SHA-256 Hash: 185f8db32271fe25f561a6fc938b2e264306ec304eda518007d1764826381969

3.15 Random Substitution Cipher: !rs

Description: Applies a random substitution cipher to the message, where each letter is replaced by another letter.

Usage: !rs Hello

Example Output:

**Random Substitution Cipher**
Text: Hello
Encrypted Message: A randomized substitution of letters

3.16 Salted Hashing: !sh

Description: Generates a salted hash by adding a random salt to the message and hashing it using SHA-256.

Usage: !sh Hello

Example Output:

**Salted Hash**
Text: Hello
Salted Hash: [hashed_value]:[salt_value]

4. Combo Command

4.1 Combo Encryption and Decryption: !combo

Description: This command allows you to combine multiple encryption, decryption, or hashing methods. Specify the methods in the order they should be applied, separated by commas.

Usage: !combo <method1,method2,...> <message>

Example 1: !combo ce,he HelloWorld

Applies Caesar Cipher encryption followed by Base64 encoding.

Example 2: !combo rs,sh HelloWorld

Applies Random Substitution Cipher followed by salted hashing.

Example Output:

**Combined Encryption/Decryption**
Original Text: HelloWorld
Caesar Cipher: KhoorZruog
Base64 Encoding: S2hvb3JacnVvZw==

Available Methods for Combo:

  • ce: Caesar Cipher Encryption
  • cd: Caesar Cipher Decryption
  • he: Base64 Encoding
  • hd: Base64 Decoding
  • be: Binary Encoding
  • bd: Binary Decoding
  • me: Morse Code Encoding
  • md: Morse Code Decoding
  • rs: Random Substitution Cipher
  • sh: Salted Hashing
